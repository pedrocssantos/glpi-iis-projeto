# GLPI no IIS

Projeto de implantação e documentação do GLPI em Windows Server com IIS, PHP FastCGI e GLPI Agent.

## Objetivo
Armazenar os arquivos do projeto e registrar a instalação, configuração, troubleshooting e boas práticas do ambiente.

## Estrutura
- `docs/instalacao.md`: instalação inicial
- `docs/configuracao-iis.md`: configuração do IIS
- `docs/configuracao-php.md`: configuração do PHP/FastCGI
- `docs/glpi-agent.md`: comunicação do agente com o GLPI
- `docs/erros-encontrados.md`: erros e correções aplicadas
- `config/`: exemplos de configuração
- `scripts/`: automações e comandos úteis

## Ambiente
- Windows Server
- IIS
- PHP via FastCGI
- GLPI
- GLPI Agent

## Observações
Arquivos sensíveis foram removidos ou sanitizados antes do versionamento.
